board games

We carry a few board games aboard Pino, to play with other sailors or just with each other on evenings or while at anchor.

  • Star Realms - Card-based deck building science-fiction tabletop game
  • Classic Paperbacks Memory Game - Matching game based on literary classics
  • Hanabi - Cooperative card game
  • Citadel - Strategy bluffing game
  • Double twelve dominoes - A set used to play a number of games
